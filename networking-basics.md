# Networking Basics

## What is an IP Address
An IP address is a unique identifier assigned to a device on a network.

## What is a Port
A port is a communication endpoint used by applications.

## Common Ports
- 80 → HTTP
- 443 → HTTPS
- 22 → SSH

## Protocols
- TCP → Reliable communication
- UDP → Faster but less reliable

## What I Learned
- Basic understanding of how devices communicate
- Importance of ports in network services
